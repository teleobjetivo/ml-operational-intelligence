# AI / ML Projects Pack — TeleObjetivo / OrionLab

Este paquete contiene **14 proyectos** listos para correr como demos locales (Jupyter + scripts).
Todos siguen el mismo patrón de estructura y documentación.

## Cómo usar (rápido)
1. Descomprime el ZIP.
2. En la carpeta del proyecto:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```

## Proyectos
- `p01_event_early_warning` — Detección temprana de eventos anómalos
- `p02_risk_scoring_evolutivo` — Predicción de riesgo dinámico (scoring evolutivo)
- `p03_operational_state_classifier` — Clasificación de estados operacionales
- `p04_demand_forecast_activation` — Predicción de demanda con activación
- `p05_situation_detector` — Machine Learning para “situaciones”
- `p06_timeline_prediction_engine` — Predicción de fechas/atrasos con timeline
- `p07_trend_atlas` — Atlas de tendencias (detección y storytelling)
- `p08_data_quality_sentinel` — Centinela de calidad de datos + reglas
- `p09_model_drift_monitor` — Monitor de drift y degradación de modelos
- `p10_root_cause_suggester` — Sugeridor de causa raíz para anomalías
- `p11_ticket_triage_automl` — Priorización de tickets con ML simple
- `p12_kpi_narrative_generator` — Generador de narrativa ejecutiva para KPIs
- `p13_alert_to_action_orchestrator` — Orquestación: alerta → acción
- `p14_executive_demo_dashboard` — Dashboard demo ejecutiva (HTML)
